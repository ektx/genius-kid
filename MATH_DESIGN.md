# 数学计算游戏设计文档

## 1. 风格指南 (Style Guide)
### 视觉风格：拼音风格
- **核心理念**：寓教于乐，采用明快、卡通化的视觉语言。
- **色彩规范**：
  - 主色调：`#00acc1` (青色)，象征智慧与冷静。
  - 背景色：`#e0f7fa` (浅青)，营造舒适的视觉环境。
  - 按钮色：`#ffb300` (琥珀色)，突出交互感。
  - 成功色：`#4caf50` (绿色)。
  - 错误色：`#f44336` (红色)。
- **字体与形状**：
  - 使用圆润的卡片设计（`border-radius: 24px+`）。
  - 加粗的数字与运算符，确保儿童易于识别。

## 2. 关卡设计逻辑
游戏共设计 15 个难度递进的关卡：
- **Level 1-3**：20以内基础加减法。
- **Level 4-6**：三个数加减混合运算（如 `10 + 5 - 2`）。
- **Level 7-10**：填空题（Missing Operand），如 `8 + ( ) = 15`。
- **Level 11-15**：复杂等式，两边均有运算，如 `10 + ( ) = 7 + 7`。

## 3. 功能特性
- **实时反馈**：每道题提交后立即给出动效反馈。
- **星级评分**：
  - 3星：100% 正确。
  - 2星：80% 以上正确。
  - 1星：60% 以上正确。
- **无障碍支持**：
  - 遵循 WCAG 2.1 标准。
  - 完整的 ARIA 标签支持。
  - 全局键盘操作支持（数字键、退格键、回车键）。

## 4. 部署方案
- **技术栈**：Vue 3 + TypeScript + Pinia + Vue Router。
- **存储**：使用 `localStorage` 进行本地进度保存。
- **环境要求**：支持 HTML5 的现代浏览器。
- **构建方式**：`npm run build` 生成静态文件，可部署至任意 Web 服务器。

## 5. 测试报告
- **响应式测试**：已在 PC 端、平板及移动端通过适配测试。
- **逻辑测试**：随机生成的题目确保结果始终为非负整数，无死题。
- **兼容性**：支持 Chrome, Edge, Safari 等主流浏览器。
